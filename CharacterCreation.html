Character Creation

1. Roll Up The Character

  1. Roll or pick a Home World using the table below.
  2. Roll or pick a Background using the table below.
  3. Roll or pick a Role using the table below.
  4. Roll 2d10 + 20 for each Characteristic. For Characteristics in the bonus column for the Home World, reroll and keep the highest. For Characteristics in the penalty column, reroll and keep the lowest.
  5. Roll for the number of Wounds as indicated on the Home World table.

    5. Spend Experience and Equip Gear

    1. Get 1000(?) XP.
    2. Buy any Advances you wish with this XP. Keep any XP you don't spend.
    3. [buy equipment and requisition rare items]
    4. Set your Fate Points to 3.

    Buying Advances

    Each different Advance has an XP cost and two associated Aptitudes. If you have both those Aptitudes, you can take the Advance for half the XP cost. If you have one matching Aptitude, the XP cost is as written. If you have no matching Aptitudes, the XP cost is doubled. There are three different kinds of Advance: Characteristics, Skills and Talents.

    • Characteristic Advances. Increases a Characteristic by 5. You can take this up to 5 times for each Characteristic, with the XP cost going up each time. The XP cost and Aptitudes are shown in the tables below.
    • Skill Advances. Increases the bonus for a Skill by 10, or aquire a non-basic skill at a +0 bonus. You can take this up to a +30 bonus for each skill. The XP cost is shown in the table below. The Aptitudes assicoated to each skill are listed in the Skills section.
    • Talent Advances. xxxxxx
