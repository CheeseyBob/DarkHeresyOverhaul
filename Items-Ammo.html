GM Guide For Item Creation

The base availability for Ammo is the weapon's availability +2, to a maximum of Abundant (+3). Modifiers should also be applied for special rules the item has.

Special Rules
Name Av. Description
External Ammo -1 This is designed to be used with a belt-fed weapon.
External Ammo Tank -1 This is designed to be used with a weapon with a feed line.
Rechargeable -2 This item can be recharged from a power supply.
